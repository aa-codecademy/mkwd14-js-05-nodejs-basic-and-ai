## ⚠️ Requirements for submitting homework

You **must** use a **new GitHub repository** for all Node.js homeworks (one repo for the whole course — create it specifically for this course). Add the trainer as a collaborator so your work can be reviewed.

### 1. Create a new repository

1. Go to [github.com/new](https://github.com/new).
2. Choose a name (e.g. `nodejs-homeworks`).
3. Set visibility to **Private** (recommended) so only you and the trainer can see it.
4. Click **Create repository**.

### 2. Make the repo private (if you didn’t in step 1)

1. Open your repository on GitHub.
2. Go to **Settings** → **General**.
3. In **Danger Zone**, find **Change repository visibility**.
4. Click **Change visibility** → choose **Private** → confirm.

### 3. Add the trainer so he can review

1. In your repo, go to **Settings** → **Collaborators** (or **Collaborators and teams**).
2. Click **Add people**.
3. Enter **ivokostovski** and add the user with at least **Read** access.

### 4. When you’re done with a homework

**Send an email to [ivo@kostovski.dev](mailto:ivo@kostovski.dev)** with the homework title and the link to your repository. Submitting only by pushing code is not enough — the trainer has no automatic notification, so you must email when the homework is ready for review.
